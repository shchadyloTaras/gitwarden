# GitWarden Docs

GitWarden is a desktop Git app that keeps your GitHub identities straight. If you use separate accounts for personal projects, work, or client repositories, GitWarden makes the active identity always visible and blocks unsafe commits and pushes before they happen.

## What GitWarden does

- **One profile per repository** — assign each repository to a Personal, Work, or Client profile
- **Checks before every commit and push** — verifies your name, email, SSH key, and remote against the assigned profile
- **Identity always visible** — the active profile and a safety badge appear in the app header at all times
- **Connects to GitHub** — sign in with your GitHub account so pushes go through the right identity
- **Optional AI assistant** — commit message help, change review, and a chat panel; advisory only, never runs Git for you

## Quick start

1. **[Download GitWarden](/docs/installation)** — pick the installer for your OS
2. **[Install it](/docs/installation)** — there is a one-time security warning on first launch; the install guide walks through it
3. **[Create your first profile](/docs/profiles)** — open the Profiles screen and add your name and email
4. Open a repository — GitWarden shows the assigned profile and safety badge in the header
5. Stage your changes and commit — the identity is checked before the commit goes out, and again before any push
